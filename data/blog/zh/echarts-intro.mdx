### 1. 最简单的一句话介绍

只需记住这三行核心代码：

```html
<div id="chart" style="width:100%; height:500px;"></div>

<script src="https://cdn.jsdelivr.net/npm/echarts@5/dist/echarts.min.js"></script>

<script>
var chart = echarts.init(document.getElementById('chart'));
chart.setOption({ /* ← 几乎所有的魔法都发生在这里 */ });
</script>
```

**99% 的精力都将花费在修改 `setOption()` 对象上**。

### 2. 2025-2026年最推荐的引入方法（选其一）

| 方法 | 代码语法 | 最适合 | 推荐程度 |
|---------|-------------|------------|----------------|
| CDN（最简单） | `<script src="https://cdn.jsdelivr.net/npm/echarts@5"></script>` | 快速原型、文档、演示 | ★★★★★ |
| 指定版本（最稳定） | `https://cdn.jsdelivr.net/npm/echarts@5.5.1/dist/echarts.min.js` | 生产项目（防止自动更新问题） | ★★★★☆ |
| 按需引入（最小打包体积） | `import * as echarts from 'echarts/core';` + 手动组件引入 | Vue/React大型项目 | ★★★★ |
| npm完整包 | `npm i echarts` → `import * as echarts from 'echarts'` | 常规前端工程项目 | ★★★☆ |

**大多数人只需要使用第一或第二种方法即可！**

### 3. 快速参考 - 常用图表模板（可直接复制粘贴）

```js
// 1. 经典柱状图
option = {
  title: { text: '产品销量对比' },
  tooltip: {},
  xAxis: { data: ['T恤', '卫衣', '裤子', '袜子'] },
  yAxis: {},
  series: [{
    name: '销量',
    type: 'bar',
    data: [120, 200, 150, 80, 70],
    itemStyle: { color: '#5470c6' }   // 更改颜色
  }]
};

// 2. 折线图（平滑 + 多条线）
option = {
  title: { text: '7日流量趋势' },
  tooltip: { trigger: 'axis' },
  legend: { data: ['桌面端', '移动端'] },
  xAxis: { type: 'category', data: ['周一','周二','周三','周四','周五','周六','周日'] },
  yAxis: { type: 'value' },
  series: [
    { name: '桌面端', type: 'line', smooth: true, data: [120, 132, 101, 134, 90, 230, 210] },
    { name: '移动端', type: 'line', smooth: true, data: [220, 182, 191, 234, 290, 330, 310] }
  ]
};

// 3. 饼图（最常见的用法）
option = {
  title: { text: '用户设备分布', left: 'center' },
  tooltip: { trigger: 'item' },
  legend: { orient: 'vertical', left: 'left' },
  series: [{
    type: 'pie',
    radius: '55%',
    center: ['50%', '60%'],
    data: [
      { value: 1048, name: '移动端' },
      { value: 735, name: '平板' },
      { value: 580, name: '桌面端' },
      { value: 484, name: '其他' }
    ],
    emphasis: {
      itemStyle: {
        shadowBlur: 10,
        shadowOffsetX: 0,
        shadowColor: 'rgba(0, 0, 0, 0.5)'
      }
    }
  }]
};

// 4. 柱状图 + 折线图组合（很常见）
option = {
  tooltip: { trigger: 'axis' },
  legend: {},
  xAxis: { data: ['1月','2月','3月','4月','5月','6月'] },
  yAxis: [
    { type: 'value', name: '销量（单位）' },
    { type: 'value', name: '增长率（%）' }
  ],
  series: [
    { name: '销量', type: 'bar', data: [300, 450, 620, 580, 790, 920] },
    { name: '增长率', type: 'line', yAxisIndex: 1, data: [5, 12, 18, -4, 35, 16] }
  ]
};
```

### 4. 快速技巧参考（强烈建议收藏）

| 功能 | 如何实现 | 备注 |
|---------|------------------|-------|
| 更改主题色 | `color: ['#5470c6', '#91cc75', '#fac858', ...]` | 全局颜色列表 |
| 暗色主题 | `chart = echarts.init(dom, 'dark');` | 第二个参数为'dark' |
| 自定义字体 | `textStyle: { fontFamily: 'Microsoft YaHei', fontSize: 14 }` | 全局字体 |
| 隐藏图例 | `legend: { show: false }` | — |
| 横向柱状图 | 交换 `xAxis` 和 `yAxis` 位置 | 超方便 |
| 柱子上显示数据标签 | `series: [{ label: { show: true, position: 'top' } }]` | — |
| 自动刷新数据 | `chart.setOption(newOption, true)` | 第二个参数true清空并重新绘制 |
| 响应式图表 | `window.onresize = () => chart.resize()` | 必须写！ |
| 渐变柱子 | `itemStyle: { color: new echarts.graphic.LinearGradient(...) }` | 稍微复杂一点 |
| 点击事件 | `chart.on('click', params => console.log(params.name, params.value))` | 超重要！ |

### 5. 推荐学习路径（建议顺序）

1. **让4种基础图表跑起来**（1-2小时）
2. **学习图例/提示框/标题配置**（30分钟）
3. **玩转颜色、字体、网格、边距**（1小时）
4. **学习dataset + transform**（新的数据驱动方式，强烈推荐！）
5. **尝试雷达图、散点图、地图、桑基图、树图**（按需）

**官方示例库（2026年仍最推荐）：**
https://echarts.apache.org/examples/zh/index.html

### 最后总结

> ECharts 不是关于 ECharts 本身
> 而是关于**你想画什么样的图表**、**你的数据是什么样的**、**你想讲什么样的故事**

一旦你有了明确的需求，找到最相似的例子，改数据 → 改标题 → 改颜色 → 加交互 → 就完成了。
