# 什么是 Markdown？

Markdown 是一种在网络上设置文本样式的办法。您可以控制文档的显示；将文字格式化为粗体或斜体，添加图片，创建列表，这些都是我们可以使用 Markdown 完成的众多事情中的几个。大多数情况下，Markdown 只是常规文本加上一些非字母字符，比如 `#` 或 `*`。

# 语法指南

以下是您可以在 GitHub.com 上任何地方或您自己的文本文件中使用的 Markdown 语法概述。

## 标题

```
# 这是一个 h1 标签
## 这是一个 h2 标签
#### 这是一个 h4 标签
```

# 这是一个 h1 标签

## 这是一个 h2 标签

#### 这是一个 h4 标签

## 强调

```
_这段文字将是斜体_
**这段文字将是粗体**
_你可以**结合**它们_
```

_这段文字将是斜体_

**这段文字将是粗体**

## 表格

您可以通过组装一系列单词并用连字符 `-`（第一行）分隔它们，然后用竖线 `|` 分隔各列来创建表格：

```
| 第一个标题                | 第二个标题                |
| --------------------------- | ---------------------------- |
| 单元格 1 的内容         | 单元格 2 的内容          |
| 第一列的内容 | 第二列的内容 |
```

| 第一个标题                | 第二个标题                |
| --------------------------- | ---------------------------- |
| 单元格 1 的内容         | 单元格 2 的内容          |
| 第一列的内容 | 第二列的内容 |

## 删除线

任何被两个波浪号包围的单词（比如 `~~这样~~`）将显示为 ~~删除线~~。
